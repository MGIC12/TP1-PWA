## Explicación de funcionalidades de archivos principales:

#### index.js:

Es el primer archivo de JS que el empaquetador procesa, ya que importa el componente App.js y hace uso de ReactDOM para inyectarlo en el dominio del navegador.

#### App.js:

Es el componente raíz de la interfaz de usuario. En este archivo se estructura visualmente todo el contenido

#### index.css:

Es la hoja de estilos global del proyecto.

#### package.json:

En este archivo se encuentran las dependencias o librerías necesarias para el proyecto, también incluye scripts para ser ejecutados en el proyecto y datos, como por ejemplo, el nombre del proyecto.

## Instalación del proyecto:

Es necesario tener instalado Node.js!!

Se hace uso de npm, ejecutando en el intérprete de comandos lo siguiente:

```bash
  npm install
  cd TP1-PWA/
```

## Ejecución del proyecto en un servidor local:

Esto se hace posible gracias a Vite.

Se ejecuta en el intérprete de comandos lo siguiente:

```bash
  npm run dev
```
